== Installation ==



= optimizeMember is very easy to install (instructions) =
1. Upload the `/optimizeMember` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Navigate to the `optimizeMember Options` panel for configuration details.


= Is optimizeMember compatible with Multisite Networking? =
Yes. optimizeMember and optimizeMember Pro, are also both compatible with Multisite Networking. After you enable Multisite Networking, install the optimizeMember plugin. Then navigate to `optimizeMember -> Multisite (Config)` in the Dashboard on your Main Site.

== Description ==


The optimizeMember system integrates with PayPal Website Payments Standard, and fully supports recurring billing. optimizeMember supports custom Pages for registration (including Custom Registration/Profile Fields), account access, and a lot more. optimizeMember is compatible with Multisite Networking too. We also have support for unlimited Membership Levels, PayPal Website Payments Pro, Authorize.Net, Google Checkout, ClickBank, advanced User Import/Export tools, the ability to use Coupon Codes, and many other enhancements.
